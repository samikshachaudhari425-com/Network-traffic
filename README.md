# Network-traffic
# 🚀 Network Packet Analyzer

A Python-based **Network Packet Analyzer** built using Scapy that allows you to capture, simulate, and analyze network packets in real-time and test environments.

---

## 📖 Overview

This project helps you understand:
- Network packet structure
- Protocol behavior (TCP, UDP, ICMP, DNS, HTTP)
- Basic intrusion detection concepts

It provides **3 modes**:
1. Live Sniffing  
2. Interactive Packet Simulation  
3. URL-based Packet Testing  

---

## ✨ Features

- 🔍 Real-time packet sniffing  
- 🧪 Manual packet simulation  
- 🌐 URL to packet conversion  
- 🚨 Basic DoS attack detection  
- 📡 Protocol identification  
- 🌍 HTTP request parsing  
- 🔎 DNS query detection  
- 💾 Save packets to `.pcap` file  

---

## 🛠️ Technologies Used

- Python  
- Scapy  
- Socket  
- Datetime  
- urllib
